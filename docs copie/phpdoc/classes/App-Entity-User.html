Class User

Représente un utilisateur dans l'application, avec un nom d'utilisateur, un mot de passe haché et des rôles. Cette entité implémente UserInterface et PasswordAuthenticatedUserInterface, ce qui la rend compatible avec le système de sécurité de Symfony.
